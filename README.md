## 简介

**Forme** [fɔ:m]，意为模板。但这并不是取这个名字的本意。或许拆开理解为 **for me** [fɔ: mi:] 更为贴切。

希望每个使用该模板的人都有种 "just for me" 的感觉，同时能轻松拿到心仪的多金的Offer。
